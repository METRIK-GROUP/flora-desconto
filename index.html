Como o desconto funciona

O desconto é aplicado na hora do plano escolhido. No plano anual ele vale o ano inteiro. No plano mensal vale só o primeiro mês, e depois volta ao valor normal.

Dúvidas que podem aparecer

O desconto não apareceu. O que faço?

Quase sempre é um destes dois motivos:

  • Você não se inscreveu pelo link de parceiro.
  • Você usou um email que já era cadastrado no Flora.

Solução: refaça o passo a passo usando um email novo. Cogitamos pedir um cupom para adicionar no checkout, mas por questões internas do Flora não é possível. Então refaça a operação e, se o erro persistir, tente trocar de navegador.

Testamos aqui com vários emails e funcionou a partir de 30/09/2025. Se você teve esse problema antes dessa data, refaça o cadastro.

Não sei se invisto direto no plano anual.

Se quiser testar a ferramenta primeiro, assine um plano de 1 mês com o link. Depois desse período, use outro email para assinar o anual, também pelo link.

Já tenho conta no Flora. Consigo o desconto?

Não na conta atual. O desconto só vale para uma conta nova, com email novo, criada pelo link de parceiro. Se você tem um plano vigente, use o link quando ele vencer.

O desconto vale para qualquer plano?

Sim, os 25% valem para todos os planos. No anual o desconto vale o ano inteiro; no mensal, vale só o primeiro mês e depois volta ao valor cheio.
